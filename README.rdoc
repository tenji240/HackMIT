Selector - Readme
---------

Ruby version - 2.1.2p95

System dependencies
-------------------
  * PostgreSQL
  * Rails v4.1.4

Basic Setup
------------
  * if Postgres isn't installed
    * `brew install postgresql`
    * `initdb /usr/local/var/postgres -E utf8`
  * `psql`
  * `CREATE ROLE drugview LOGIN SUPERUSER;`
  * `bundle install`
  * `rake db:create`
  * `rake db:setup`
  * `rails server`
